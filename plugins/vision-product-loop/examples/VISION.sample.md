# Product Promise

Help a builder turn an uncertain product idea into a working product through a crisp vision, project-aware planning, and repeated evidence-based implementation loops.

## Users

- Product-minded builders who want an agent to help define and ship an idea.
- Engineering teams that already have local architecture, agents, and validation harnesses.

## Must-Have Capabilities

- The agent must find or create a durable product vision before major implementation work begins.
- The agent should grill unclear intent until acceptance criteria are concrete enough to test.
- Users can bring any repo architecture, agent setup, or harness without rewriting the loop.
- Each implementation loop must compare real evidence against the agreed vision.

## Acceptance Checks

- A user can invoke the skill to transform a vision document into a slice-by-slice implementation loop.
- The loop identifies project-local tests, demos, screenshots, or evaluators before claiming completion.
- Remaining gaps are recorded after each loop.

## Non-Goals

- This plugin does not prescribe a specific framework, issue tracker, or agent runtime.
- This plugin does not replace project-specific architecture docs.
